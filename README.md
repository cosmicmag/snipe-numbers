# snipe-numbers — +888 анонимные номера, снайп-арбитраж

Пилот: банкролл $3–5k, режим **полу-авто** (бот сам берёт если спред > порога, иначе алерт).
Фокус — анонимные номера. Эталон-конкурент: `btcmacho.t.me` (on-chain разбор показал
~$100–200/день, профит сидит в "купить дёшево на Getgems", не в спреде).

## Что уже есть (Phase 1, keyless, РАБОТАЕТ, в продакшене через cron)
- `getgems_feed.py` — **KEYLESS Getgems-фид** через wallet-API `api.getgems.io/wallet/collectiblesOnSale/{coll}`
  (пускает по `origin: walletbot.me`, БЕЗ ключа). Это дешёвая BUY-витрина. ~190 fix-price TON листингов.
- `fragment_feed.py` — keyless-скрапер Fragment (number, price, тип, таймер). Sell-витрина / флор.
- `pattern_value.py` — fair-value по паттерну цифр (repeat/palindrome/seq/zeros/low-distinct).
- `alert_bot.py` — два источника → сигналы в Telegram. Дедуп по `state.json`.
- `run.sh` — раннер с env (токен внутри). Запускается cron каждые 5 мин → `cron.log`.

Запуск вручную: `./run.sh`  ·  один тик. Диагностика стакана: `python3 getgems_feed.py 300`

## Сигналы бота
- 🎯 **RARE-SNIPE** — паттерн (mult≥1.4: seq3+/тройки/палиндром/нули) дешевле pattern-fair на MARGIN+
- 💰 **CROSS-VENUE ARB** — Getgems fix-price → Fragment, NET после комсы (~10%) ≥ MIN_NET_TON
- 📉 **FLOOR-DROP** — Getgems fix-price ниже прежнего флора
- 🆕 **NEW-CHEAP** — новый fix-price в топ-5 дешёвых

## Ключевой факт (проверено на живых данных 2026-06-04)
Getgems fix-floor ~1550–1668, Fragment floor ~1688. Сырой спред ~120 TON, НО после комсы 10% =
**в минус** → CROSS-VENUE сейчас НЕ срабатывает (честно: чистого арба между витринами нет, комса ест).
Реальный edge — ловить ОТДЕЛЬНЫЕ недооценённые листинги (rare-паттерн / резкий демп ниже флора),
а не системный спред витрин. Бот именно это и сторожит.
